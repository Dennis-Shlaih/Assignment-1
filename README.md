# Assignment 1: Personal Portfolio Website

A responsive personal portfolio website built using HTML and CSS to showcase my projects, skills, and experience.

## Overview

This portfolio serves as a central place to highlight my work and skills. It is designed to be simple, responsive, and easy to navigate across all devices.

## Features

- Responsive design for desktop, tablet, and mobile
- Clean multi-section layout:
  - Home / Intro
  - About Me
  - Projects showcase
  - Skills section
  - Contact information
- Flexbox-based layout system
- Consistent color theme and styling
- Smooth navigation between sections

## Built With

- HTML5
- CSS3

## How to Run
Open `index.html` in any browser.
